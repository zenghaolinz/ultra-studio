# Ultra Studio 0.7.0 Generation Task Center Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build an event-driven, restart-safe generation task center with per-task cancellation and retry while keeping chat responsive and `chat.py` from growing.

**Architecture:** SQLite remains authoritative; mutation services publish normalized task snapshots through an in-process broker and SSE route. Tauri forwards one Sidecar stream to a shared Zustand store consumed by chat cards and the queue center.

**Tech Stack:** Python 3.10/FastAPI/aiosqlite, Rust/Tauri 2/reqwest, React 18/TypeScript/Zustand 5, Python unittest.
